## Running the Scripts
Describe any specific dependencies, libraries, or environment configurations required to run your scripts.
---
Project was run from a virtual environment using python 3.12 on Ubuntu. This distribution was managed by conda.
`conda create -n venv python=3.12` (to create the repository)
`conda activate venv` (to run code from the virtual envrionment)
---
Missing libraries had to be installed with pip

Aside from this, running the test_submission code for either file is as easy as running the file in the venv
`python T1/script.py` or
`python T2/script.py`
